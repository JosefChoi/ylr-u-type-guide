Safety Interlock 및
Remote Start 변경 (U type)

PHOENIX CONTACT 12-PIN CONNECTOR
01
Connector Change: Harting 7-Pin (Old) → Phoenix Contact 12-Pin (New)
02
Pin 1-4: Power Supply Active Ch 1/2 (Status Output)
03
Pin 5-6: Remote Key Switch (Contact Closure, 1–5s) 이전 메인에서 이동
04
Pin 7-8: Remote Start Button (Momentary Contact, 1s) 이전 메인에서 이동
05
Pin 9-12: Interlock Door A/B (Dual Channel Loop)
🔧 Action Item: E-Stop 및 외부 시작 버튼 배선을 반드시 이 커넥터로 새로 구성해야 합니다.

  • Fiber 케이블 외경: 6.8mm → 8.6mm
